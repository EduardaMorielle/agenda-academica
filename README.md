# Agenda Acadêmica Web

Aplicação web para divulgação e organização de eventos acadêmicos, desenvolvida com HTML, CSS e JavaScript puro.

## Funcionalidades

- Listagem dinâmica de eventos em cards
- Pesquisa por título em tempo real
- Filtro por categoria
- Cadastro de novos eventos com validação
- Destaque visual para eventos com poucas vagas (≤ 5)
- Indicação de eventos encerrados
- Layout responsivo com Grid e Flexbox
- Validações HTML5 e JavaScript
- Estrutura semântica e acessibilidade

## Tecnologias Utilizadas

- HTML5 semântico
- CSS3 (Grid, Flexbox, media queries)
- JavaScript ES6+ (DOM, eventos, arrays)
- Git e GitHub para versionamento

## Como usar

1. Clone o repositório.
2. Abra o arquivo `index.html` em um navegador moderno.
3. Use o campo de pesquisa para filtrar eventos por título.
4. Use o filtro de categoria para exibir apenas eventos específicos.
5. Cadastre novos eventos pelo formulário.

## Deploy

O projeto foi feito para ser publicado no GitHub Pages.

1. Crie o repositório no GitHub.
2. Faça o push para a branch `main`.
3. No GitHub, abra `Settings` → `Pages`.
4. Selecione a branch `main` e a raiz do projeto `/`.
5. Salve e aguarde a publicação.
